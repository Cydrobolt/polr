Welcome to Polr

You can download the latest release by clicking the tar.gz or zip buttons. Polr is currently in the Beta stage, therefore, some things may still be buggy. If you are having any issues with Polr, please click hereto join our support channel. You can also read the docs here. If you would like to contribute, click the View on Github button at the top right corner.

About Polr

Polr is a minimalist URL shortening framework written in PHP. However, you can easily integrate it into any web application that supports HTTP GET. Consult the wiki page for examples in Python, PHP, and Node.js.

IRC Channel

Polr uses IRC for communication. We are on Freenode (irc.freenode.net:6667) at #polr.

Polr URL Shortener

A "live demo" can be viewed here: Polr.cf, the official URL shortener created by Polr developers. You can use this URL shortener any way you want. It is not solely for demo purposes, but rather is a full-blown URL shortener.
